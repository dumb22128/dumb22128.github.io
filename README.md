# Сайт — Герман Олейник, психолог

Одностраничный сайт-визитка практикующего психолога. Экзистенциально-гуманистический подход, онлайн.

- **Стек:** один статический `index.html` (HTML + CSS + крошечный inline-JS, ваниль). Фото вшито как data-URI — внешних ассетов нет.
- **Хостинг:** GitHub Pages.
- **Контакт:** [Telegram](https://t.me/pokureynik) · [ВКонтакте](https://vk.com/teslapanda) · [Instagram](https://instagram.com/pokureynik)

## Локальный просмотр
Просто открыть `index.html` в браузере (двойной клик) — всё работает без сервера.

## Деплой
Любой push в ветку `main` автоматически обновляет сайт на GitHub Pages.
Файл `.nojekyll` отключает обработку Jekyll. Настройки контента и тона — в `CLAUDE.md`.
